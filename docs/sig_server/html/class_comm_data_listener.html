説明

CommDataDecoder によるデータ復元結果取得リスナーインターフェース.

データ復元時に、それぞれCommDataTypeに対応したメソッドがコールバックされる。

  1. 第1引数 fromはデータの転送元
  2. 第2引数 evt には復元したデータが含まれる(データを含まないシグナルだけの呼び出しもある)
