説明

CaptureViewImageサービス要求.

コントローラから受け取ったスクリーンショット撮影依頼に関する情報を保持するクラス．

スクリーンショットの撮影に必要なエージェント名、必要な画像のサイズ、フォーマット等の情報を保持する．

この情報は、メインスレッドのサービス要求キューに送られ、 メインスレッドによりスクリーンショットが撮影される．

メインスレッドは撮影した画像をCaptureViewImageServiceResultに格納し サービス結果キューに入れる．
