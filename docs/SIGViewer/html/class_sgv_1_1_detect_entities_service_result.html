説明

DetectEntitiesサービス結果.

メインスレッドによるエンティティの探索結果

メインスレッドは、発見したエンティティの名前（複数）を このオブジェクトに格納し、サービス結果キューに入れる．

サービスプロバイダスレッドはサービス結果キューから このオブジェクトを取り出し、サービス要求元（＝コントローラ）に返信する．
