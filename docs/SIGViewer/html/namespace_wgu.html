コントロールの位置、サイズを変える

アンカーフラグにより、ウィンドウの端に「くっついた」状態を 簡単に表現できる．

引数:
hwndParent [i] 親ウィンドウ
controlId [i] コントロールId
x1 [i] 横パラメータ1
x2 [i] 横パラメータ2
y1 [i] 縦パラメータ1
y2 [i] 縦パラメータ2
anchorLeft [i] 左アンカーフラグ
anchorRight [i] 右アンカーフラグ
anchorTop [i] 上部アンカーフラグ
anchorBottom [i] 下部アンカーフラグ
x1, x2, y1, y2の意味は、アンカーフラグによって以下のように変わる．

[anchorLeft=false, anchorRight=falseの場合] x1=左端からの距離、x2=コントロールの幅

[anchorLeft=true, anchorRight=falseの場合] 上に同じ

[anchorLeft=false, anchorRight=trueの場合] x1=コントロールの幅、x2=右端からの距離

[anchorLeft=true, anchorRight=trueの場合] x1=左端からの距離、x2=右端からの距離

[anchorTop=false, anchorBottom=falseの場合] y1=上端からの距離, y2=コントロールの高さ

[anchorTop=true, anchorBottom=falseの場合] 上に同じ

[anchorTop=false, anchorBottom=trueの場合] y1=コントロールの高さ, y2=下端からの距離

[anchorTop=true, anchorBottom=trueの場合] y1=上端からの距離, y2=下端からの距離
