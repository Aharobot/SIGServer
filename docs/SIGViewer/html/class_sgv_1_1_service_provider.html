bool Sgv::ServiceProvider::sendNSPingerResult ( bool  active  ) 

PINGを打ってきた相手に返答する.

引数:
active [i] true(=サービス受け付けok), false(=サービスは受け付けられない)
戻り値:
true(=送信成功), false(=送信失敗)
SIGVerseのプロトコルの取り決めにより、 シミュレーションサーバは、 コントローラからのサービス受け付け要求を受けると、 サービスを提供するはずのサービスプロバイダが 実際に「生きているか」を調べるために PINGをサービスプロバイダに送ってくる．

このメソッドはそのPING要求に返答するものである．
