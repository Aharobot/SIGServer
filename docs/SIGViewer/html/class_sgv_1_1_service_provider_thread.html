スレッドで回すサービスプロバイダを与える．

run()によりスレッドが開始されると、 スレッドループは、与えられたサービスプロバイダのloopingProc()を呼び出し続ける．

サービスプロバイダのloopingProc()がfalseを返すとスレッドループは終了する．
