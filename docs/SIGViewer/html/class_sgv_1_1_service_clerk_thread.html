サービスクラークスレッド

サービスクラークをスレッドで実行する．

メインプログラムはこのスレッドを起動し、外部からのサービス要求を監視する．

スレッドで回すサービスクラークを与える．

run()によりスレッドが開始されると、 スレッドループは、与えられたサービスクラークのloopingProc()を呼び出し続ける．

サービスクラークのloopingProc()がfalseを返すとスレッドループは終了する．
