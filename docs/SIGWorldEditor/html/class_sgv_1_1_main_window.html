enum Sgv::MainWindow::QueryFlags

Ogreクエリーマスクフラグ.

OgreエンティティにsetQueryFlags()メソッドでフラグを設定しておくと RaySceneQuery等のクエリーを行うときに、検索対象に含めたり 外したりの選択をすることができる．

SELECTION_TARGET  クエリー検索対象にする
