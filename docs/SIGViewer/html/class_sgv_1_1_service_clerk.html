説明

サービスクラーク

外部からのサービスの受付を行う． 受け付けたサービスは、サービスプロバイダに処理を委譲する．

サービスクラークは、実行中のサービスプロバイダのリストを保持している． 処理の終了したサービスプロバイダは、サービスプロバイダのリストから削除される．

int Sgv::ServiceClerk::accept (  ) 

コントローラからのサービス要求を受け付ける

サービス要求受付用ソケットに、コントローラからの接続要求が 来ているかをチェックする．もし来ているなら、サービスプロバイダスレッドを 起動する．

このメソッドは、サービスクラークスレッドのloopingProc()から 定期的に呼び出される．

戻り値:
 > 0 ... サービスリクエスタからのサービス受付要求があり、処理された
 = 0 ... サービスリクエスタからのサービス受付要求はなかった
 < 0 ... エラー
