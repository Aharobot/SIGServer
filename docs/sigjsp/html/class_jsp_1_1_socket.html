int Jsp::Socket::recvPacket ( Jsp::Packet **  ppPacket  ) 

データをパケットで受信する

引数:
ppPacket [o] 受信したパケットを受け取るポインタ
戻り値:
1(=成功), -1(=失敗)
このメソッドはSIGVerseのパケットの開始、終了マークを判定し、 1つのパケットが完全に届くまで待つ．

受け取ったパケットはppPacketに返ってくる． 呼び出し側はppPacketの解法責任を持つ． （すなわち、呼び出し側はppPacketをdeleteしなければならない）
