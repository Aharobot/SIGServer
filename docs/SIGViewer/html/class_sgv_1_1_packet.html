説明

パケット（ネットワーク通信の単位となるブロックデータ）

ネットワークを流れるSIGVerseのデータはすべて、 開始マーク(=0xABCD)で始まり、 終了マーク(=0xDCBA)で終わるデータブロックで構成されている． このデータブロックをパケットと呼ぶ．
