説明

X3DまたはVRMLファイルをエンティティの形状としてロードするクラス.

javaライブラリXj3Dにより、X3DまたはVRMLファイルをパースし、 その結果をエンティティの形状として保存する。

環境変数SIGVERSE_X3DPARSER_CONFIGにより、Xj3D使用時の javaオプションを記述したファイルX3DParser.cfgを指定できる
