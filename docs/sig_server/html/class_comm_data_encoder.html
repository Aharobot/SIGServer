説明

転送データエンコーダー基本クラス

各転送データ毎にこのクラスを継承したエンコーダークラスを作成し、 そこで転送データタイプに応じて分割パケット数の指定(packetNum関数)、 データのバイナリ化(encode関数)を実装する。
