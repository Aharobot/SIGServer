ビューアのインストール

準備


以下の説明では，ディレクトリ名の指定で次の略記を使います.
[SIGVERSE_PACKAGE]
SIGVERSEインストールパッケージのトップディレクトリ．
例えば、配布パッケージ中のbin, doc, srcディレクトリは以下のように表記されます．

[SIGVERSE_PACKAGE]/bin
[SIGVERSE_PACKAGE]/doc
[SIGVERSE_PACKAGE]/src

ビューアのインストール

ビューアのインストーラは以下のファイルです．

[SIGVERSE_PACKAGE]/bin/SIGViewerSetup/setup.exe

このファイルをダブルクリックするとインストーラが起動します．




インストーラは以下の順序で必要なコンポーネント（ライブラリ）をビューア本体をシステムにインストールします．

  • DirectX 9.0cランタイム
  • Visual C++ 2005 SP1再配布可能ファイル
  • OpenALライブラリ
  • JRE(Java Runtime Environment) 1.6.0
  • ビューア本体

    それぞれの項目について以下に説明します．

    DirectX 9.0cランタイム

    DirectX 9.0cのランタイムをシステムにインストールします．




    DirectX 9.0cランタイムがシステムにインストールされていない場合、もしくはインストールされているか不明な場合は「Yes」を選択してください．すでにDirectX 9.0cランタイムがシステムにインストールされている場合は「No」を選択してください．

    （注）Windows 7またはWindows Vistaの場合、システムに最初からDirectX 10.0以降が入っているため、この段階をスキップしても構いません．

    「Yes」を選んだ場合、次のようなダイアログが現れます．指示に従ってDirectX 9.0cランタイムをインストールしてください．





    Visual C++ 2005 SP1再配布可能ファイルのインストール

    Visual C++ 2005ランタイムをシステムにインストールします．




    「Yes」を選択した場合、以下のようなダイアログが出ます．しばらくしてこれが消えればインストールは終了です． （「インストール完了」のようなメッセージは出ませんが，問題ありません）




    OpenALライブラリのインストール

    次にOpenALのランタイムライブラリをシステムにインストールします．




    「Yes」を選択した場合、以下のダイアログが表示されますので、「OK」を押してください．これでインストールが実行されます．




    JRE 1.6.0のインストール





    JRE1.6.0以上がシステムにインストールされていない場合は，「Yes」を選択します．システムにすでにJRE1.6.0以上がインストールされている場合は、「No」を選択します

    「Yes」を選択した場合、JREのインストーラが起動しますので、指示に従ってインストールしてください．




    （注）場合によってはOpenOfficeのインストールを行うか否かのチェックボックスが出ることがあります. しかしSIGViewerはOpenOfficeは使用しませんので、インストールする必要はありません

    JREのトップディレクトリの指定


    次に、SIGViewerが使用するJREの場所を指定します．




    インストーラは、入力されたJREの場所を基準に、jvm.dllファイルを探します． jvm.dllが見つからない場合はメッセージが出ますので、再度JREの場所を指定しなおしてください．


    ビューア本体のインストール

    次にビューア本体をインストールします．インストール先ディレクトリを入力し、「Next」を押してください． （インストール先ディレクトリは任意です）





    いくつかのオプション設定ダイアログ（アイコンをデスクトップ上に作成するか？等）が出た後、以下のようなダイアログが表示されます．「Install」をクリックするとビューア本体のコピーが開始されます．





    インストール完了

    次のダイアログが表示されればインストールは完了です





    ビューアの使用方法については、「ビューアの起動」をご覧ください）.





    ビューアのアンインストール

    Windowsのコントロールパネルから「プログラムの追加と削除」を開き、「SIGViewer 1.0」をアンインストールします．

    アンインストールの確認ダイアログが出ますので、本当にアンインストールしてよければ「はい」を押してください





    以下のダイアログが出ればアンインストールは完了です
