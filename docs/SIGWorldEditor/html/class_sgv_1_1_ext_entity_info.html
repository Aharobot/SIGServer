説明

Ogreエンティティに持たせる追加情報.

Ogreの各エンティティはOgreUserDefinedObjectから派生したクラスの オブジェクトを保持することができる． SIGViewerでは、このしくみを用いて、各Ogreエンティティに補助的情報を持たせている．
