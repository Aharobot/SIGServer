説明

CaptureViewImageサービス結果.

メインスレッドにより撮影された、 エージェント視点からのスクリーンショットを保持するクラス

メインスレッドは、撮影した画像をこのオブジェクトに格納し、 サービス結果キューに入れる．

サービスプロバイダスレッドはサービス結果キューから このオブジェクトを取り出し、サービス要求元（＝コントローラ）に返信する．
