説明

X3Dファイルデータベースクラス.

エージェントのパーツ、状態に応じたX3Dファイルを管理する。 キー文字列は 


    <エージェント名>:<パーツ名>:<状態名>:<状態値>

パーツ名、状態名、状態値はなくてもキーとして使用できる。
