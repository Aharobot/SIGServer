int CCSV::CountSeqCells ( int  irow,
int  icol,
int  drow,
int  dcol 
)

(irow, icol)から(drow, dcol)ステップで移動して、 値のあるセルの個数を数える

引数:
irow [i] サーチ開始行
icol [i] サーチ開始列
drow [i] 行方向ステップ値
dcol [i] 列方向ステップ値
戻り値:
連続セルの個数
      A, B, C
      1, 2
      3,  , 4

    というcsvファイルがあったとすると、

      CountSeqCells(0, 0, 0, 1) --> 3   /// Aから右へサーチ
      CountSeqCells(0, 1, 1, 0) --> 2   /// Bから下へサーチ
      CountSeqCells(2, 0, 0, 1) --> 1   /// 3から右へサーチ
      CountSeqCells(2, 1, 0, 1) --> 0   /// 空白からサーチ
      CountSeqCells(10, 10, 0, 1) --> 0 /// 領域外からのサーチ

    ＃ちなみにLoadFromFile()でbTrim=trueなら空白の連続は
    すべて飛ばされるので、カンマの間にいくつスペースを入れてもそのセルには
    何も入っていないとみなされる

    (ex)
     A,B,,C
     と書いても
     A,B,        ,C
     と書いても
     CountSeqCells(0, 0, 0, 1) --> 2
