説明

DetectEntitiesサービス要求.

コントローラから受け取ったエンティティの探索依頼に関する情報を保持するクラス．

この情報は、メインスレッドのサービス要求キューに送られ、 メインスレッドによりエンティティの探索が行われる．

メインスレッドは探索結果をDetectEntitiesServiceResultに格納し サービス結果キューに入れる．
