 SSimObjBuilder (SSimObj &o, ODEWorld &w)

説明

エンティティオブジェクトに必要な要素を設定するクラス

エンティティにシミュレーションに必要な形状や属性等を設定する。

コンストラクタで設定対象となるSSimObjを与えておく． addJoint(), addParts()等のメソッドは、コンストラクタでセットされたSSimObjに 大してジョイントを追加したり、パーツを追加したりする．
