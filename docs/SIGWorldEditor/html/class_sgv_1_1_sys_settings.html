説明

設定情報の保存用クラス

SIGWorldEditor終了時に、サーバー名、ポート番号等の現在の設定を保存するために使われる．

Sgv::StartupFileと似ている．異なるのは以下の点．

Sgv::StartupFileは読み込み専用．設定の保存はできない． Sgv::SysSettingsは読み書き両方ができる．
