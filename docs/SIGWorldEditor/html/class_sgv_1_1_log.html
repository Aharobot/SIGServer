void Sgv::Log::addListener ( LogListener *  l  ) 

ログリスナー（＝ログの出力先）を追加する

引数:
l [i] ログリスナー
print()等のメソッドで渡されたログメッセージは このメソッドで登録したログリスナーにより出力される． ログリスナーは複数個登録できる．

ファイルログ(FileLogListener)を登録した場合、ログはファイルに出力される． GUIログ(GUILogListener)を登録した場合、ログはGUI部品(エディットコントロール）に出力される．

登録したログリスナーの所有権はこのオブジェクトに移る． すなわちこのオブジェクトの破棄時に、登録されているすべての ログリスナーはdeleteされる．

 Log l;
 l.addListener(new FileLogListener());  // ファイルログを追加
 l.addListener(new GUILogListener());   // GUIログを追加
 ...
 l.print("hello");  // ファイルとエディットコントロールの両方に出力される
