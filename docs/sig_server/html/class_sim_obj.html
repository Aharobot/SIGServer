エンティティに対応するクラス(クライアントサイドで使用)

シミュレーションサーバから対応するエンティティの情報を取 得し、クライアントサイドでそれらにアクセスできるようにす る。SimObjに対する操作・属性等はシミュレーションサーバに 転送され、その結果が反映される。
