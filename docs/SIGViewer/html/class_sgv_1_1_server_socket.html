int Sgv::ServerSocket::accept ( Socket **  ppSocket  ) 

acceptする

クライアントソケットからの接続を待つ． クライアントからの接続要求がある場合、acceptは成功する． さらにクライアントとの間の通信用に新しいソケットが生成され、ppSocketに返る．

クライアントからの接続要求がない場合、ブロッキングモードならば接続が来るまでブロックする．ノンブロッキングモードの場合、0を返し即座にリターンする．

戻り値:
= 1 ... accept成功．生成された新しいソケットはppSocketに返る．

= 0 ... タイムアウトした（ノンブロッキングモードの場合のみ）

= -1 ... エラー発生
