X3Dオブジェクトからノードを作る.

引数:
mgr [i] Ogreシーンマネージャ
parentNode [i] 親ノード
px3d [i] X3Dオブジェクトへのポインタ
stateValue [i] 状態値
visObjElemName [i] この静止物につける名前
scale [i] スケール（拡大）係数
bUseX3DCenterOfMass [i] OpenHRPのCenterOfMassフィールドを使用するか否かのフラグ
ss [i] 単純形状オブジェクト．NULLを与えた場合はX3D形状から自動生成する．
bShowSimplifiedShape [i] 単純形状の表示スイッチ
戻り値:
true(=成功), false(=失敗)
parentNodeの子ノードを生成し、 X3Dオブジェクトを元に属性（位置、方向など）をセットする．

ノードの階層構造は以下のようになる

parentNode(=引数) +--visObjElemNode .................. (1) +--NNode (=通常形状ノード) .... (2) +--SNode (=単純形状ノード) .... (3)

生成されたノード(上記の(1), (2), (3))は呼び出し元には帰らない． （単にparentNodeの子ノードが増えるだけ） ノードを操作する必要がある場合は、getVisObjNode(), getShapeNode(), getSimplifiedNode() のいずれかを使い必要なノードを取得してから行う．
